E-Mail-Versand (Massenmail)

Direktversand der Rundmails über Google Apps Script – verschickt aus eurem Vereins-Google-Konto. Einmalig einrichten (Anleitung unten), dann kannst du im Tab „Massenmail" direkt senden.

So richtest du den Versand ein (einmalig)
  1. Mit dem Vereins-Google-Konto auf script.google.com → „Neues Projekt".
  2. Den Code unten komplett einfügen (vorhandenes ersetzen) und speichern.
  3. „Bereitstellen" → „Neue Bereitstellung" → Typ Web-App, Ausführen als Ich, Zugriff Jeder → Bereitstellen, Berechtigungen erlauben.
  4. Die Web-App-URL (endet auf /exec) oben einfügen und „Verbindung testen".

Verschickt wird über das Google-Konto, mit dem du das Skript erstellst. Die „Absender-Adresse" wird als Antwort-Adresse gesetzt (und als Absender, falls sie ein verifizierter Alias dieses Kontos ist).

function doPost(e){
  var d = JSON.parse(e.postData.contents);
  if (d.action === 'test'){ return out({ ok:true }); }
  if (d.action === 'mail'){
    var to = d.from || Session.getEffectiveUser().getEmail();
    var opts = { bcc:(d.bcc||[]).join(','), name: d.fromName || 'ELKI Verein Trimmis', replyTo: d.from || '' };
    try { if (d.from) opts.from = d.from; GmailApp.sendEmail(to, d.subject || '(kein Betreff)', d.body || '', opts); }
    catch (err){ delete opts.from; GmailApp.sendEmail(to, d.subject || '(kein Betreff)', d.body || '', opts); }
    return out({ ok:true, sent:(d.bcc||[]).length });
  }
  return out({ ok:false });
}
function out(o){ return ContentService.createTextOutput(JSON.stringify(o)).setMimeType(ContentService.MimeType.JSON); }

Daten Import/Export

Dokumente am Kind

PDF- und Bild-Dateien (PNG/JPG) werden pro Kind direkt in der Datenbank (Firestore) gespeichert – keine zusätzliche Firebase-Einrichtung nötig, es funktioniert sofort. Die Dokumente bleiben über alle Schuljahre erhalten.

Max. 10 MB pro Datei. Grosse Dateien werden automatisch in Teile zerlegt. Hochladen/Löschen erfolgt im Kind-Formular (Bereich „Dokumente"); ein Klick auf den Dateinamen öffnet das Dokument.

Schuljahre / Zeitachsen

Zeitachsen umbenennen, löschen oder neu anlegen. Beim Umbenennen werden alle Zuteilungen mitverschoben. Beim Löschen gehen die Zuteilungen dieses Jahres verloren – die Kinder und ihre Dokumente bleiben erhalten.
